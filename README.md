# Outfit Recommender

An AI based outfit recommender app that learns from user style preferences to progressively suggest outfit combinations they are more likely to fancy.

# FEATURES:
1. Streamlit dashboard for user interaction
2. Gender Based Filtering
3. Image Represenation through embeddings/one hot encoding through tensorflow's ResNet50
4. Machine learning model that learns from user likes/dislikes

# TECH STACK:
1. Python
2. Streamlit
3. Pandas
4. NumPy
5. RandomForestClassifier
6. CLIP (for labeling clothes' gender)
7. Tensorflow

To run locally:
pip install -r requirements.txt
streamlit run app.py
